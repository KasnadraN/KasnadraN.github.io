Create a diagram with 2 classes. Those classes need 3 attributes, 3 methods
```mermaid
    classDiagram
    class Fin{
        -length
        -width
        -height
        -weight
        +stability()
        +direction()
        +flight()
    }

    class NoseCone{
        -length 
        -width 
        -height
        -weight
        +stability()
        +direction()
        +prevent rocket from slowing down()
    }
```
Attributes come before properites or methods<br>
\+ public, \- private, \# protected<br>
